Overview

This project is a data analysis project using Python, analyzing my Spotify streaming data and creating visualizations.

Getting Started

Prerequisites

Before you can get started with this project, you need to have Conda installed on your machine. If you don't have Conda installed, you can download and install it from the official Conda website.
